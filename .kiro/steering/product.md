# Product Overview

This is a personal portfolio website for Shajida Akter Lopa, a MERN Stack Developer. The site showcases professional experience, skills, projects, and provides a contact form for potential clients and collaborators.

## Key Features

- Hero section with animated typewriter effect
- About section with personal journey and background
- Skills showcase
- Education timeline
- Projects gallery with detailed project pages
- Contact form with EmailJS integration
- Smooth scroll navigation
- Responsive design for all devices
- Dark-themed UI with purple accent colors

## Target Audience

Potential employers, clients, and collaborators looking to learn about the developer's skills and work.
